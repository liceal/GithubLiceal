1.先查看下服务器仓库或者远程仓库
git remote -v

2.获取最新的版本到本地
git fetch origin master

3.比较本地和远程仓库代码的差别
git log -p  master.. origin/master

4.远程下载的代码和本地合并
git merge origin/master

5.本地上传全部（暂译）
git -u origin/master

6.修改本地推送到的远程仓库地址
git remote add origin 地址

7.将本次更改更新到github项目上去
git push -u origin master

8.add全部内容
git add .

9.同步仓库全部内容到本地
git pull origin master
